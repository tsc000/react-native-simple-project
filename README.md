# react-native-simple-project
一个简单的工程（只为展示基本的写法）



1. **Clone the repo**

  ```
  $ git clone https://github.com/BelinChung/react-native-hiapp.git
  $ cd react-native-hiapp
  ```

2. **Install dependencies** (npm v3+):

  ```
  $ npm install
  ```

3. **Start Packager:**

  ```
  $ npm start
  ```

4. **Running on iOS:**

  ```
  $ react-native run-ios
  ```
